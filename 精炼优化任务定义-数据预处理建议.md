# 数据预处理建议
## 1. 缺失值处理

轻度缺失（<5%）：均值/中位数填充（数值型）、众数填充（分类型）
中度缺失（5-20%）：KNN插补、随机森林插补
重度缺失（>20%）：考虑删除该特征或使用领域知识推断
## 2. 异常值检测与处理

方法：3σ原则（正态分布）、IQR方法（箱线图）、DBSCAN聚类
处理：直接删除明显错误值、用上下限截断、标记后单独分析
## 3. 数据标准化/归一化

数值型特征：StandardScaler（标准化）或MinMaxScaler（归一化到[0,1]）
注意：树模型（LightGBM、XGBoost）对特征尺度不敏感，可跳过此步骤
## 4. 特征工程

时序特征：历史炉次滞后特征（上炉温度、过热度）、滑动窗口统计量
比率特征：合金加入量/钢水量、渣料加入量/钢水量、合金收得率
交互特征：合金元素含量×氧活度（反映脱氧效率）
钢种编码：One-Hot编码或嵌入编码（Embedding，用于神经网络）
## 5. 数据分割策略

训练集/验证集/测试集：70%/15%/15% 按时间顺序分割
交叉验证：时序交叉验证（TimeSeriesSplit）或留出法
避免数据泄露：确保同一炉次的数据只出现在一个集合中

# 数据扩充建议
## 1. 滑动窗口方法

目的：从多采样炉次中创建多个训练样本
方法：对于有N次采样的炉次，取前k次采样作为输入，第k+1次作为标签
效果：可将数据量放大3-5倍
## 2. 机理数据增强

目的：补充极端工况或稀有炉种的数据
方法：基于热力学方程生成合成样本，作为补充训练数据
注意：合成数据需标注，训练时降低其权重
## 3. 数据增强策略

时间扭曲：微调时间戳（±1-2分钟）用于模拟测量偏差
噪声注入：在温度、成分等测量值上添加高斯噪声（模拟测量误差）
# 数据分析建议
## 1. 探索性数据分析（EDA）

单变量分析：各特征的分布直方图、统计量（均值、方差、分位数）
双变量分析：特征与目标的相关性散点图、相关性热力图
多变量分析：PCA降维、t-SNE可视化（用于异常检测）
## 2. 时序分析

趋势分析：温度、成分随时间的变化曲线
周期性分析：不同班次、不同日期的工艺参数差异
异常检测：基于统计过程的控制图（SPC）
## 3. 特征重要性分析

方法：模型训练后的特征重要性排序（LightGBM的feature_importance、SHAP值）
目的：识别关键工艺参数，指导优化策略
